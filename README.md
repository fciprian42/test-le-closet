# Challenge Le Closet

## Lancement

Lancer le back (port 3000) :

```bash
cd server
bundle install
rails db:setup
rails s
```

Lancer le front (port 3001) :


```bash
cd client

npm config set "@fortawesome:registry" https://npm.fontawesome.com/
npm config set "//npm.fontawesome.com/:_authToken" 9DED4BD8-824B-4EFE-8079-0BF0A05E17CB

npm install
npm start
```
